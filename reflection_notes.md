# Reflection Notes – DeepThought Assignment

## 1. Approach
- I built a deterministic decision tree using Python.
- I used nested dictionaries to store questions and answers.
- The program asks Yes/No questions and always gives the same output for the same input.

## 2. Guardrails Against AI Hallucination
- I did not let AI generate random answers.
- I hardcoded fixed rules so the tree is predictable.
- I checked the logic myself to make sure it makes sense.

## 3. Disagreements with AI
- AI suggested using machine learning, but I chose a simple deterministic tree.
- I avoided unnecessary complexity because the assignment asked for deterministic logic.

## 4. Negative Prompting
- I told AI not to use randomness or probability.
- I asked for clear, rule-based solutions only.

## 5. Alignment with Guidelines
- Followed the instructions in the `.md` file.
- Made sure the tree is reproducible and structured.
- Prepared documentation (README.md) and kept the repo professional.
